OPTION DE L’INTERFACE

Pour les calculs des états selon les règles de l’évaluation SEEE :

Cette interface permet de choisir les états à évaluer. Il est possible de déterminer uniquement un ou plusieurs états « intermédiaires », à savoir :

·         L’état physico-chimique

·         L’état des polluants spécifiques

·         L’état biologique

·         L’état chimique

L’état écologique est automatiquement calculé si l’utilisateur choisit de calculer l’état physico-chimique et l’état biologique.

Le très bon état est déterminé sans tenir compte de l’état hydromorphologique et l’état des MEFM n’intègre pas les CTO (contraintes techniques obligatoires).

Il est nécessaire de les prendre en compte à postériori pour confirmer le très bon état et l’état écologique des MEFM.

·         Contamination aigüe
