Cette interface permet de définir des options de calcul avancées.

Les options de calculs sont liées aux choix précédents. Les options cochées ou décochées par défaut correspondent aux règles de l’arrêté.

Pour le module de contamination, les options sont quelques peu différents. L’interface permet en autre de définir les coefficient à appliquer aux seuils pour

OPTIONS DE CALCUL AVANCEES :

-          Assouplir le très bon état

En activant cette option, le très bon état sera assoupli. Il peut être assoupli si l’état biologique est très bon et qu’un seul paramètre physico-chimique est bon (les autres étant très bons). Cela reprend les principes d’assouplissement du bon état.

-          Simulation ¼ de classe

En activant cette option, on simule l’assouplissement des éléments de qualité biologique selon le principe du ¼ de la classe. A priori cette option sera obligatoire à partir du 3ème cycle.

-          Exclure les polluants non synthétiques de l’état PS

En activant cette option, les polluants non synthétiques sont calculés mais exclus de l’état final des polluants spécifiques, car sans les valeurs de fond géochimique ils sont régulièrement responsables du déclassement de la station.

-          Compléter la PCH avec des données modélisées

En activant cette option, il est possible d’intégrer un fichier de données contenant des valeurs de paramètres physico-chimiques issues d’une modélisation (PEGASE notamment). Les valeurs peuvent être les résultats bruts de la modélisation ou les classes d’état. A défaut de PCH sur une station alors qu’il y a des données biologiques, cela permet de compléter les données et définir l’état écologique (règle permise dans l’arrêté).

-          Extrapoler l’état à la masse d’eau (état écologique et/ou état chimique et/ou Contamination)

En activant cette option, l’état écologique et/ou indépendamment l’état chimique des stations seront extrapolés à la masse d’eau. Cela suppose de connaître la ou les stations représentatives sur la masse d’eau et de connaître l’association station-masse d’eau.

Si plusieurs stations représentatives possèdent le même état pour qualifier la masse d’eau, les données finales de chaque station seront exportées. L’utilisateur devra déterminer la station retenue pour qualifier la masse d’eau.

Il est également possible de définir des stations temporairement représentatives, à défaut de données sur les stations représentatives. S’il y a des données sur les stations représentatives et temporairement représentatives, la station représentative sera prioritaire pour l’extrapolation de l’état à la masse d’eau.

 

-          PCH / PS / CHIMIE / CONTAMINATION : Respecter la fraction

En désactivant cette option, il est possible des traiter les paramètres sur le support « eau », mais ne respectant pas la fraction définie dans l’arrêté (eau brute ou eau filtrée). Cette option concerne indépendamment les paramètres physico-chimiques, les polluants spécifiques et les paramètres chimiques.

La table des codes fraction par paramètre est consultable dans le dossier d’installation : MON_DOSSIER_S3R\PARAMETRES\DEFAUT\PARAMETREXXX.csv

-          PCH / PS / CHIMIE/ CONTAMINATION  : Respecter l’unité des paramètres

-          En désactivant cette option, il est possible des traiter les paramètres sans tenir compte de leur unité. Cette option concerne indépendamment les paramètres physico-chimiques, les polluants spécifiques et les paramètres chimiques.

La table des unités est consultable dans le dossier d’installation : MON_DOSSIER_S3R\PARAMETRES\DEFAUT\PARAMETREUNITE.csv

-          PCH / PS / CHIMIE/ CONTAMINATION  : Respecter le nombre minimal de prélèvements

En désactivant cette option, il est possible des traiter les paramètres sans tenir du nombre minimal de prélèvements imposés (4 prélèvements minimum). Cette option concerne indépendamment les paramètres physico-chimiques, les polluants spécifiques et les paramètres chimiques.

-          PS / CHIMIE : Méthode – calcul de la moyenne

L’utilisateur peut choisir l’une des méthodes proposées pour calculer la moyenne annuelle des polluants spécifiques et des paramètres chimiques :

·         L’année la plus récente présente dans le jeu de données pour chaque paramètre sera retenue pour calculer la moyenne annuelle (règle imposée dans l’arrêté).

·         L’année ayant la meilleure fréquence de prélèvement dans le jeu de données pour chaque paramètre sera retenue pour calculer la moyenne annuelle.

 

-          PS / CHIMIE : Retrait des prélèvements (Cycle 2 et )

Les prélèvements (analyses) sont retirés du calcul si la LQ est supérieur à la NQEMA et si la valeur moyenne mesurée est inférieur à la LQ.
